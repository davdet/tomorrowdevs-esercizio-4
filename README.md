# ESERCIZIO 4 Frontend Integration
- Aprire l'Esercizio 1 e seguire le istruzioni per farlo partire
- Aprire l'Esercizio 4  con VS Code o editor equivalente e installare i pacchetti npm digitando `npm i` all'interno di un terminale
- Far partire il progetto digitando dentro un terminale `http-server -p 8080`
- Aprire un browser e digitare `httplocalhost8080` nella barra degli indirizzi
- Interagire con l'interfaccia

## Cosa fa l'esercizio
L'esercizio crea un front-end rudimentale ma funzionante per il back-end creato nell'Esercizio 1

## Scelte tecniche
Nessuna scelta tecnica in particolare da segnalare. Come da direttive l'esercizio è stato eseguito in vanilla JavaScript senza l'ausilio di framework. A differenza degli altri esercizi, dove ho fatto del mio meglio per mantenere un codice quanto più pulito possibile, qui la cosa non è stata altrettanto semplice e il codice, soprattutto nel file `app.js`, potrebbe risultare a tratti confusionario, seppur funzionante.
